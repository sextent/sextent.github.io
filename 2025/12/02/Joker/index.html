3.Method

输入与输出

我们的方法以一张肖像图像作为输入,创建一个具有真实感的3D重建结果。驱动表情通过Basel Face Model[32]的参数和文本提示来指定(见图2)。

我们训练了一个条件2D先验模型,该模型:

  • 输入:参考人物的图像
  • 输出:预测具有新表情的新视角渲染结果（3.1节）

利用这个2D先验及其预测结果,我们提出了一个新颖的3D蒸馏管线（3.2节）来优化视图一致的NeRF。

3.1 用于极端表情合成的2D先验模型

我们的先验模型基于**Stable Diffusion[53]**骨干网络。

条件合成模型的构建

为了将其转换为条件合成模型,我们遵循[7, 9, 25, 72]的做法,训练一个独立的参考网络,将参考图像的信息输入到去噪网络中。
具体来说,我们在参考网络的自注意力层与其去噪对应层之间共享键(keys)和值(values)[25]。
然后我们训练一个ControlNet[78](我们称之为驱动网络),用于3DMM参数条件控制。与之前的方法[25, 72]相比,并类似于并行工作[9],我们使用基于网格的法线图渲染来编码驱动头部姿态和表情。我们观察到,这种条件控制方式有助于实现结果的更高视图一致性,即使没有使用视图感知噪声或多视图自注意力技术[25]。

文本条件控制的创新

与现有工作[4, 9, 72]不同,我们还保留了基础去噪模型的文本条件,并将其整合到参考网络和驱动网络中。
文本条件使我们的模型能够通过补充3DMM信号中缺失的线索来合成极端表情,例如:

  • 舌头运动
  • 与情感相关的外观细节

此外,与之前基于扩散的重演方法相反，我们发现微调去噪网络的解码部分有助于改善身份保持。

为了训练这个模型,我们实施了一个迭代训练管线:

  1. 使用野外CelebV-Text数据集[74]的子集进行训练
  2. 随后在NeRSemble数据集[37]上进行短期微调

我们为这些数据集补充了新的元数据,包括:

  • 表情的文本标注
  • 3DMM拟合参数(详见附录A)

3.2 3D蒸馏

遵循[56, 65]的做法,我们使用NeRF[49]公式来表示我们的3D重建。
在蒸馏过程中,NeRF在多个目标视角下进行渲染,并被编码到扩散模型的潜在空间中。
优化步骤

  1. 向这些渲染结果添加噪声
  2. 2D扩散先验对潜在表示进行去噪
  3. 将去噪后的潜在表示解码为目标图像 ˆx₀
  4. NeRF针对这些目标图像进行优化

蒸馏损失函数

遵循之前的工作[56, 65, 70],我们在图像域而非潜在空间中直接应用蒸馏损失。
优化目标遵循[70],由渲染图像x和目标图像ˆx₀之间的L1距离和**感知损失Lp[79]**组合而成:

$$ L_{recon} = E_c|x - \hat{x}_0|_1 + L_p(x, \hat{x}_0) (1)$$

其中,期望值是对不同视角c计算的。

两阶段蒸馏程序

我们的蒸馏程序包含两个阶段,它们在目标图像更新方式上有所不同:

  • 阶段1: 目标图像基于NeRF渲染结果反复更新,以提高多视图一致性
  • 阶段2: 目标图像仅预测一次,然后NeRF针对这些目标图像优化直至收敛

阶段1:动态目标优化

在第一阶段,我们使用噪声化的NeRF渲染结果作为蒸馏先验的输入,频繁更新目标图像。

  1. 在多个目标视角下渲染NeRF
  2. 对渲染结果添加噪声
  3. 使用2D先验通过一步去噪生成新的目标图像
  4. NeRF针对目标视图优化N次迭代,之后我们重新计算目标并重复该过程。我们使用3D一致的NeRF渲染来更新目标图像,以提高其视图一致性。

与之前的方法[56, 65, 70]相比,主要区别在于:

  1. 确定性噪声采样: 我们遵循标准的DDIM去噪调度器,确定性地采样噪声级别(100步),而不是随机采样
  2. 更新频率: NeRF在更新目标图像之前针对目标图像优化N次迭代,而大多数现有方法在每次NeRF更新时都更新目标

我们发现这显著提高了重建结果的视觉质量和一致性(见图6)。

阶段2:固定目标优化

仅执行阶段1蒸馏往往会导致模糊的结果。
原因在于NeRF针对目标图像的优化过程:

  1. 在优化过程中,NeRF实际上对目标图像中的不一致性进行平均,引入了低频偏差
  2. 这对蒸馏程序产生重大影响
  3. NeRF优化后,其渲染结果会比目标图像更模糊
  4. NeRF渲染中缺乏高频细节被2D先验捕获,并传播到更新的目标图像中

3.3 实现细节

2D先验模型训练

基础配置

  • 起始权重: Stable Diffusion v1.5¹的预训练权重
  • 优化器: AdamW[47]
    • 学习率 lr = 10⁻⁵
    • β₁ = 0.9, β₂ = 0.999
    • ϵ = 10⁻⁸, λ = 10⁻²

训练环境

  • 硬件: 8块NVIDIA A100 SXM4-80GB
  • 批量大小: 每GPU 10个样本
  • 训练迭代:
    • 在CelebV-Text[74]上训练200,000次迭代
    • 在CelebV-Text和NerSemble[37]的均匀混合数据上额外训练30,000次迭代

数据集和预处理详情请参见附录A。

3D蒸馏配置

框架与参数

  • 框架: threestudio[27]
  • 配置: 主要遵循ImageDream[65]的配置

目标图像生成

  • 在正面半球的20×20规则网格上渲染BFM[32]法线图
  • 方位角范围: ∈ [−22.5°, 22.5°]
  • 仰角范围: ∈ [−10°, 10°]
  • 无分类器引导尺度: 19.0

NeRF优化

  • 每次目标图像更新之间优化N = 130次迭代
  • 优化过程中随机采样64个64×64大小的图像块
  • 目标图像分辨率从64逐渐增加到512
  • 优化时间: 在单块NVIDIA A100 SXM4-80GB GPU上约需3小时
